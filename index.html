Tag Checker

"Tag Checker" is a headless application built using AWS services (primarily a Step Function state machine) in order to iterate through all AWS services created in an account and verify that the tagging methodology of developers spinning up resources are meeting tagging standards. If tagging standards aren't met, the resource is deleted and a record is logged in an RDS db for further functionality.
